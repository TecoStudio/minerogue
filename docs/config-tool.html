Roguelike YAML Helper

配置片段助手

这是一个静态页面，直接双击打开即可使用。填写字段后复制生成的 YAML 片段，放入对应的 plugins/Roguelike/<file>.yml，再在用户自己运行的服务器中执行 /rw reload 或用 PlugManX 重载 Roguelike。

使用顺序

  • 先选模板，再改字段和效果。
  • 只复制当前片段到对应文件；相同 ID 会覆盖已有模板。
  • 概率使用 0.0-1.0，例如 0.12 表示 12%。

武器模板

生成 `weapons.` 片段。效果字段会原样写入 `effects`，常见键包括 `attack_range`、`crit_chance`、`crit_damage`、`lifesteal_percent`、`fire_damage`、`fire_duration`、`slow_duration`、`slow_level`、`chain_targets`、`chain_range`、`chain_damage_percent`、`lightning_chance`、`poison_chance`、`poisoned_target_damage_percent`、`dash`、`hyper`。

基础字段

目标文件：weapons.yml
复制到 plugins/Roguelike/weapons.yml

    物品模板

    生成 `items.` 片段。`item` 是物品材质字段；`potion` 和 `tonic` 默认可用 `minecraft:potion`。预设包含治疗药水、迅捷药剂和铁肤药剂。

    基础字段

    目标文件：items.yml
    复制到 plugins/Roguelike/items.yml

      精英怪 / Boss 难度

      这里调整的是已有内部精英怪的精英/Boss-like 难度参数，不会新增新的 Boss 运行时。生成路径为 `internal.skeleton-elite`、`internal.zombie-elite` 或 `internal.spider-elite`。

      内部精英怪字段

      目标文件：mobs.yml
      复制到 plugins/Roguelike/mobs.yml

        普通怪物词条/难度

        生成 `modifiers.` 片段，用于普通怪物生命、攻击、速度倍率和生成时使用的 Roguelike 武器模板。

        普通怪强化字段

        目标文件：mobs.yml
        复制到 plugins/Roguelike/mobs.yml

          提示：本页只生成可复制的 YAML 片段，不会读取或修改任何服务器文件。修改 plugins/Roguelike/weapons.yml、items.yml 或 mobs.yml 后，在用户自己运行的服务器中执行 /rw reload，或用 PlugManX 重载 Roguelike。
