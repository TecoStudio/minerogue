# Minerogue 游玩说明

本文档说明当前插件的主要游玩内容。内容基于当前代码实现，不包含尚未实现的设想功能。

## 核心循环

1. 玩家击杀怪物获得 Roguelike 经验。
2. 玩家升级后获得强化券、开发券、移除券等奖励。
3. 使用开发券把普通物品开发成 Roguelike 武器，或给已有 Roguelike 武器添加词条。
4. 使用强化券提升武器或防具词条。
5. 使用移除券移除不想要的词条，并为武器下次普通强化提供成功率加成。
6. 持续刷怪、获取经验、强化装备，形成 Roguelike 成长循环。

## 券系统

### 强化券

`ticket_a`，显示名为“强化券”。

作用：打开 GUI，让玩家选择要强化的武器词条。成功率随使用次数递减。失败会消耗强化券，并为下次普通强化增加 3%-13% 成功率。

普通强化券成功率：

| 已使用次数 | 基础成功率 |
| ---: | ---: |
| 0 | 100% |
| 1 | 100% |
| 2 | 95% |
| 3 | 90% |
| 4 | 85% |
| 5 | 80% |
| 6 | 70% |
| 7 | 60% |
| 8 | 50% |
| 9 | 25% |
| 10+ | 按 `0.5^(useCount - 8)` 继续衰减 |

### 超级强化券

`super_ticket_a`，显示名为“超级强化券”。

作用：打开同样的强化选择 GUI，但强化必定成功。

### 开发券

`ticket_b`，显示名为“开发券”。

作用：

| 目标 | 效果 |
| --- | --- |
| 非 Roguelike 普通物品 | 开发为 special_weapon 特殊品质武器，保留原版附魔 |
| Roguelike 武器 | 打开 GUI 确认后随机添加一个新武器词条 |
| 防具 | 打开 GUI 确认后随机添加一个防具词条 |

### 移除券

`ticket_c`，显示名为“移除券”。

作用：

| 目标 | 效果 |
| --- | --- |
| Roguelike 武器 | 打开 GUI 选择一个插件词条移除，并为下次普通强化券成功率增加 10% |
| 防具 | 打开 GUI 选择一个插件防具词条移除 |

防具移除只移除插件记录的词条。如果玩家原本拥有同类型原版附魔，移除后会恢复原来的原版附魔等级。

### 武器开发券

`weapon_development`，显示名为“武器开发券”。

作用：将任意非空气、非券、非 Roguelike 武器的物品开发为特殊品质武器。

## 原版附魔兼容

当前插件允许 Roguelike 武器同时拥有原版 MC 附魔和插件词条。

兼容规则：

| 系统 | 规则 |
| --- | --- |
| 附魔台 | Roguelike 武器不再被插件禁止附魔 |
| 铁砧 | Roguelike 武器不再被插件禁止合并附魔 |
| lore | 武器 lore 会显示原版附魔区块 |
| 插件随机词条 | 当前插件随机词条不再写入原版附魔 |
| 防具词条 | 插件防具词条不再使用原版附魔作为效果载体 |

### 铸造台

铸造台用于合成插件防具、工具和武器。

制作方式：在铁砧下方放一个白色羊毛，即可把该铁砧作为铸造台使用。

```text
铁砧
白色羊毛
```

右键铸造台会打开箱子 GUI。GUI 中间按工作台形式摆出 3x3 输入格，右侧为结果槽。当前已接入炸药套配方。

### 炸药套铸造配方

炸药套每个部件都需要 4 个 TNT，全套共需要 16 个 TNT。`C` 表示铜锭，`T` 表示 TNT，空格表示空槽。

炸药头盔：

```text
T C T
C C C
T C T
```

炸药胸甲：

```text
C T C
T C T
C T C
```

炸药护腿：

```text
T C T
C   C
T C T
```

炸药靴子：

```text
C   C
T   T
T   T
```

```text

/rl tickets
/rl trade
/rl help
```

```text

/rw debug <on|off|status>
/rw affixes
/rw reload

/rw give armor <id> [玩家] [数量]
```
