# 等级与经验

[返回目录](README.md)

## 玩家数据

玩家数据包含击杀、死亡、总经验和各类券使用次数。

## 经验来源

| 来源 | 规则 |
| --- | --- |
| 击杀怪物 | 按怪物配置立即获得经验，见 [怪物系统](mobs.md) |
| 挖矿 | 累计挖指定矿物数量，达到里程碑时获得经验 |
| 吃东西 | 累计食用指定食物数量，达到里程碑时获得经验 |

挖矿和吃东西分别独立累计。

| 累计次数 | 奖励经验 |
| ---: | ---: |
| 10 | 100 |
| 20 | 200 |
| 50 | 300 |
| 100 | 500 |
| 100 以后 | 每累计再满 100 次获得 500 |

计入挖矿累计的矿物包括煤矿、铜矿、下界石英矿、铁矿、金矿、下界金矿、红石矿、青金石矿、钻石矿、绿宝石矿、远古残骸及对应深层矿物。

计入吃东西累计的食物包括苹果、面包、胡萝卜、土豆、浆果、海带、曲奇、蜂蜜瓶、生肉/生鱼、熟肉/熟鱼、金苹果、金胡萝卜、炖菜、南瓜派和附魔金苹果等。

## 默认经验需求

| 等级段 | 到下一级经验 |
| --- | ---: |
| 1 -> 2 | 10 |
| 2 -> 3 | 20 |
| 3 -> 4 | 50 |
| 4 -> 5 | 80 |
| 5 -> 6 | 100 |
| 6 -> 7 | 150 |
| 7 -> 8 | 200 |
| 8 -> 9 | 300 |
| 9 -> 10 | 400 |
| 10 -> 11 | 500 |
| 11 -> 12 | 600 |
| 12 级以后 | 每级需求继续按代码公式递增 |

## 升级奖励

| 奖励 | 获取规则 |
| --- | --- |
| 强化券 | 每升 1 级获得 1 张 |
| 开发券 | 升到 2 级，或等级为 3 的倍数时获得 |
| 移除券 | 升到 2 级，或等级为 5 的倍数时获得 |

## 死亡惩罚

| 当前等级 | 惩罚 |
| --- | --- |
| 1-4 | 等级保护，不掉级 |
| 5-10 | 死亡掉 1 级 |
| 11+ | 死亡掉当前等级约三分之一，至少 1 级 |
