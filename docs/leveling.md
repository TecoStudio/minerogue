## 玩家数据

玩家数据包含击杀、死亡、总经验和各类券使用次数。

## 默认经验需求

| 等级段 | 到下一级经验 |
| --- | ---: |
| 1 -> 2 | 10 |
| 2 -> 3 | 20 |
| 3 -> 4 | 50 |
| 4 -> 5 | 80 |
| 5 -> 6 | 100 |
| 6 -> 7 | 150 |
| 7 -> 8 | 200 |
| 8 -> 9 | 300 |
| 9 -> 10 | 400 |
| 10 -> 11 | 500 |
| 11 -> 12 | 600 |
| 12 级以后 | 每级需求继续按代码公式递增 |

## 升级奖励

| 奖励 | 获取规则 |
| --- | --- |
| 强化券 | 每升 1 级获得 1 张 |
| 开发券 | 升到 2 级，或等级为 3 的倍数时获得 |
| 移除券 | 升到 2 级，或等级为 5 的倍数时获得 |

## 死亡惩罚

| 当前等级 | 惩罚 |
| --- | --- |
| 1-4 | 等级保护，不掉级 |
| 5-10 | 死亡掉 1 级 |
| 11+ | 死亡掉当前等级约三分之一，至少 1 级 |
