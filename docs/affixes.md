# 词条数据

## 武器/工具/弓词条

`/rw affixes` 会列出当前运行时词条。武器词条包括基础属性、吸血、连锁、暴击、状态、协同增伤、爆发/范围、主动/击杀、耐久/工具、弓/弩和契约类词条。

`dash` 不再是武器词条；它是防具词条。

## 防具词条

| ID | 名称 | 来源/定位 |
| --- | --- | --- |
| `dash` | Dash！ | 普通防具词条；空中潜行突进，默认 2 充能/5 秒冷却 |
| `thorns` | 荆棘 | 荆棘套自带；按件数反伤 |
| `swift` | 神速 | 神速套自带；每件提供速度，4 件强化 Dash |
| `explosive` | 炸药 | 炸药套自带；受击概率爆炸 |
| `guardian` | 守护 | 守护套自带；每件 6% 插件减伤 |
| `vampire` | 猩红 | 猩红套自带；击杀回血 |
| `storm` | 雷暴 | 雷暴套自带；攻击概率雷击 |
| `protection` | 保护 | 原版 Protection 附魔入口 |

## 与原版附魔重复的处理

保留：`protection`、`thorns`、`swift`。其中 `thorns`/`swift` 是套装自定义语义，不是原版荆棘/迅捷潜行附魔入口。

删除/不进入随机池：`projectile_protection`、`fire_protection`、`feather_falling`、`mending`、`efficiency`、`fortune`。

## 强化显示

- 百分比词条显示为百分比。
- 等级词条显示为等级或数量。
- 契约类词条通常同时给强增益和“受到伤害 200%”副作用。
- 防具套装自带词条在物品 lore 中显示为 `自带词条: 名称 (id)`。
