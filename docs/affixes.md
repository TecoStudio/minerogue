# 词条数据

[返回目录](README.md)

本文档列出当前代码中注册的全部插件词条。数值范围表示开发券随机生成基础值的范围。强化规则按当前代码实现说明。

## 通用规则

| 类型 | 说明 |
| --- | --- |
| 武器词条 | 默认适用于武器。由于插件中镐、斧等工具也可作为武器使用，大部分武器词条也可作用于工具。 |
| 工具限定词条 | 仅工具目标有效。 |
| 通用词条 | `Target.ALL`，武器和工具都可出现。 |
| 防具词条 | 独立防具系统，使用插件数据记录词条等级。 |

## 强化规则

| 词条类型 | 强化规则 |
| --- | --- |
| 普通小数/百分比词条 | 当前值乘以 `1.1 + 1.4 * random^2` |
| 概率词条 | 强化后最高按 100% 限制 |
| 整数词条 | 通常每次 +1 |
| 等级词条 | 每次 +1，不超过最大等级 |
| 开关词条 | 多数不可强化 |
| 中性契约词条 | 固定效果，不可强化 |

## 防具词条

防具词条通过插件数据记录等级，不再使用原版附魔作为随机词条效果载体。

当前没有注册防具随机词条。

## 与原版附魔重复的插件词条

当前没有与原版附魔重复的插件随机词条。
