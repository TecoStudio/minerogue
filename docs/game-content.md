# 游戏内容总览

## 服务器循环

Roguelike 面向自由生存服：主城负责出生、菜单、商店、教程和服务；玩家出城后自由圈地、建家、探索和刷怪。插件提供装备成长、等级和怪物压力，不限制玩家路线。

核心循环：击杀/挖矿/进食获得经验，升级获得券，使用券开发和强化装备，通过防具套装、铸造配方、怪物掉落和 Boss 事件持续成长。

## 非目标

插件不内置经济、商店定价、领地、传送或复杂任务系统；这些由 Essentials、Vault、GriefPrevention、CommandPanels、Quests 等外部插件组合。
