# 游戏内容总览

[返回目录](README.md)

本文档面向服主、管理员和玩家，集中说明 Roguelike 当前包含的游戏内容、玩法循环、获取方式和各系统之间的关系。更细的数值表和配置格式见各模块文档。

## 插件定位

Roguelike 是一个 Paper 服务端成长玩法插件，围绕“获得经验、升级拿券、开发装备、强化词条、刷怪掉落、铸造套装”的循环展开。

插件提供：

- Roguelike 玩家等级、经验、死亡惩罚和升级奖励。
- 可成长的 Roguelike 武器与词条系统。
- 强化券、超级强化券、开发券、移除券。
- 内置武器、内置物品和防具套装。
- 铸造台与可配置铸造配方。
- 普通怪经验、内置精英怪和随机武器掉落。
- 战斗伤害公式显示、侧边栏数据展示和可选外部插件集成。

插件不提供经济、货币、余额、商店或定价系统。装备和券的交易方式由服务器规则或玩家自行决定。

## 核心玩法循环

1. 玩家击杀怪物、挖矿、吃东西获得 Roguelike 经验。
2. 经验达到要求后升级，并获得强化券、开发券、移除券等奖励。
3. 玩家使用开发券把普通物品开发成特殊武器，或给已有 Roguelike 装备添加随机词条；武器不再限制随机词条槽位数量。
4. 玩家使用强化券提升武器或防具词条，失败后积累下一次普通强化成功率加成。
5. 玩家使用移除券删除不想要的词条，调整装备构筑方向。
6. 玩家通过怪物掉落、管理员发放、铸造台配方或服务器交易获取更多装备。
7. 更强装备提升刷怪效率，继续获得经验和资源，形成长期成长。

## 玩家成长

玩家数据记录击杀、死亡、总经验、券使用次数、挖矿累计和吃东西累计。

### 经验来源

| 来源 | 玩法说明 |
| --- | --- |
| 击杀怪物 | 击杀普通怪或 Boss 后按怪物类型获得经验，默认值可在 `mobs.yml` 中调整。 |
| 挖矿 | 挖指定矿物会累计次数，达到 10、20、50、100 等里程碑时获得经验。 |
| 吃东西 | 食用指定食物会累计次数，达到相同里程碑时获得经验。 |

计入挖矿累计的矿物包括煤、铜、铁、金、红石、青金石、钻石、绿宝石、远古残骸、下界石英和对应深层矿物等。

计入吃东西累计的食物包括苹果、面包、胡萝卜、土豆、浆果、海带、曲奇、蜂蜜瓶、生肉、生鱼、熟肉、熟鱼、金苹果、金胡萝卜、炖菜、南瓜派和附魔金苹果等。

### 升级奖励

| 奖励 | 获取规则 | 主要用途 |
| --- | --- | --- |
| 强化券 | 每升 1 级获得 1 张 | 强化武器或防具词条。 |
| 开发券 | 升到 2 级，或新等级为 3 的倍数时获得 | 开发普通物品，或给 Roguelike 装备添加随机词条。 |
| 移除券 | 升到 2 级，或新等级为 5 的倍数时获得 | 移除不需要的词条。 |

### 死亡惩罚

| 当前等级 | 死亡结果 |
| --- | --- |
| 1-4 | 等级保护，不掉级。 |
| 5-10 | 死亡掉 1 级。 |
| 11+ | 死亡掉当前等级约三分之一，至少掉 1 级。 |

## 战斗内容

Roguelike 的战斗以原版 Minecraft 攻击为基础，再叠加插件武器属性、词条效果、套装效果和原版附魔。

### 武器基础属性

Roguelike 武器模板包含以下核心数据：

| 属性 | 说明 |
| --- | --- |
| 物品类型 | 对应原版物品 ID，也可在启用 Nova 集成后使用 `nova:<namespace>:<id>`。 |
| 伤害 | 武器基础攻击伤害。 |
| 攻击速度 | 武器基础攻击速度。 |
| 耐久 | 武器自定义耐久。 |
| 品质 | 影响掉落、强化成功率和强化成长区间。 |
| 默认效果 | 武器生成时自带的插件词条或属性。 |

当前品质包括 `common`、`rare`、`epic`、`legendary`、`special`。

### 伤害公式显示

玩家战斗时，聊天栏会显示彩色伤害公式，例如：

```text
伤害: 7.0 + 3.5 x 2.00 x 1.50 = 31.5
```

显示内容会区分基础伤害、原版跳劈/附魔加成、契约增伤、猛击倍率、状态增伤、插件暴击、火焰附加、连锁伤害、中毒、雷击和爆炸等来源。鼠标悬停公式可查看完整计算明细。

### 特殊武器开发

玩家可使用开发券，把普通物品开发为特殊品质 Roguelike 武器。

开发普通物品时会使用 `special_weapon` 作为模板，但会保留被开发物品的关键属性：

- 原物品类型。
- 原版附魔。
- 原物品伤害。
- 原物品攻击速度。
- 原物品已有攻击距离属性。

这使普通武器、工具或其他可用物品可以进入 Roguelike 成长体系。

## 词条内容

词条是装备构筑的核心。不同词条可以提供吸血、连锁、暴击、火焰、雷击、爆炸、位移、耐久返还、矿物高亮等能力。

### 武器与工具词条方向

| 方向 | 代表词条 | 玩法效果 |
| --- | --- | --- |
| 续航 | 吸血百分比、吸血固定值、馈赠 | 通过攻击或击杀回复生命。 |
| 群伤 | 连锁目标、连锁范围、连锁伤害 | 单次攻击波及周围敌人。 |
| 爆发 | 暴击率、暴击倍率、猛击、伤害存储 | 提高单次攻击或周期性爆发伤害。 |
| 元素 | 火焰伤害、燃烧时长、中毒概率、雷电概率 | 对目标施加火焰、中毒或雷击效果。 |
| 控制 | 减速时长、减速等级 | 降低目标移动能力。 |
| 爆炸 | 爆炸概率、大爆炸概率、小心炸弹 | 触发不破坏方块小爆炸或更危险的大爆炸/TNT。 |
| 机动 | Dash、亢奋 | 提供突进、速度和急迫。 |
| 工具 | 用不坏、高亮矿物 | 降低耐久损耗，辅助挖矿。 |
| 契约 | 狂战契约、疾行契约、处刑契约等 | 获得强力固定收益，同时承担额外受伤或自损代价。 |

### 中性契约

中性契约是双刃词条，通常提供非常高的输出、速度、攻击速度、攻击距离、暴击、吸血、雷击或爆炸收益，但代价是受到伤害提高到 200%。其中血怒契约会让攻击造成更高伤害，但每次命中会自损最大生命值。

### 强化规则概览

| 类型 | 强化方式 |
| --- | --- |
| 小数/百分比词条 | 按随机倍率提升，通常偏向较低成长。 |
| 概率词条 | 强化后最高限制为 100%。 |
| 整数词条 | 通常每次 +1。 |
| 等级词条 | 每次 +1，但有最大等级限制。 |
| 开关词条 | 多数不可强化。 |
| 契约词条 | 固定效果，不可强化。 |

详细词条清单见 [词条数据](affixes.md)。

## 券系统

券是玩家装备成长的主要消耗品。

| 券 ID | 显示名 | 获取方式 | 用途 |
| --- | --- | --- | --- |
| `ticket_a` | 强化券 | 升级奖励、管理员发放、服务器交易 | 打开强化 GUI，选择并强化装备词条。 |
| `super_ticket_a` | 超级强化券 | 管理员发放、服务器活动或交易 | 强化必定成功，但强化量仍按品质区间计算。 |
| `ticket_b` | 开发券 | 升级奖励、管理员发放、服务器交易 | 开发普通物品，或给武器/防具添加随机词条。 |
| `tool_ticket_b` | 工具开发券 | 管理员发放、服务器活动或交易 | 只给 Roguelike 镐或斧添加工具类词条。 |
| `ticket_c` | 移除券 | 升级奖励、管理员发放、服务器交易 | 移除武器或防具上的插件词条。 |

普通强化券有成功率。使用次数越多，基础成功率越低；失败会消耗券，并为下次普通强化提供额外成功率。移除武器词条也会为下次普通强化提供成功率加成。

武器品质会影响普通强化券成功率和强化量。普通品质更容易强化，传奇品质更难强化，但装备基础强度和稀有度更高。

武器 lore 不再显示“流派”标签。开发券已经移除随机词条槽位数量上限，限制改为“不能重复添加已有词条；所有可用词条都拥有后才无法继续开发”。

插件接管原版经验条作为法力条：经验等级数字显示当前法力值，经验条显示法力百分比。Roguelike 等级和经验仍通过 `/rl status`、侧边栏和占位符展示。

## 防具套装

插件本体默认不在 Java 代码中内置防具显示定义。默认内容包示例位于 `content/armor/`；其中包含 3 套防具，每个 YAML 对应一个部件定义。

| 套装 | 部件 ID 前缀 | 品质 | 材质 | 玩法定位 |
| --- | --- | --- | --- | --- |
| 荆棘套 | `thorns_` | common | 锁链 | 受击反伤，4 件时受击获得亢奋与力量 II。 |
| 神速套 | `swift_` | rare | 黄金 | 每件提供速度等级，自带用不坏 IV，4 件强化 Dash 充能和冷却。 |
| 炸药套 | `explosive_` | epic | 铜质，若版本无铜防具则回退皮革 | 受击概率触发范围爆炸，自带爆炸保护 IV，4 件爆炸后获得力量 III。 |

套装效果按穿戴件数生效。荆棘套和炸药套在玩家被生物攻击时触发；神速套提供持续速度，并会影响 Dash 词条表现。

## 铸造台

铸造台用于合成插件防具、工具和武器。

制作结构为铁砧下方放置白色羊毛：

```text
铁砧
白色羊毛
```

右键该铁砧会打开铸造 GUI。GUI 中间是 3x3 输入格，右侧是结果槽。关闭 GUI 时会返还输入材料。

铸造配方存储在：

```text
plugins/Roguelike/forge-recipes.yml
```

默认文档中列出了炸药套配方。服主可以通过配方配置，把原版材料铸造成插件防具、插件武器或普通材料。

## 怪物内容

怪物经验、内置精英怪/Boss 参数和普通怪强化规则同样从 YAML 内容包加载。默认示例位于 `content/mobs/`，服务器运行时放在 `plugins/Roguelike/content/mobs/`。当前推荐一个 YAML 对应一个怪物相关定义：`type: experience`、`type: modifier`、`type: internal` 或 `type: settings`。代码只保留 `template:` 与 `actions:` 可调用的通用动作实现，具体怪物 ID、别名、属性和动作编排由 YAML 定义。

### 普通怪经验

默认普通怪经验如下：

| 怪物 | 经验 |
| --- | ---: |
| 僵尸 | 15 |
| 骷髅 | 15 |
| 苦力怕 | 20 |
| 蜘蛛 | 12 |
| 末影人 | 30 |
| 烈焰人 | 22 |
| 监守者 | 350 |
| 末影龙 | 500 |
| 凋灵 | 300 |
| 其他怪物 | 10 |

### 内置精英怪

插件包含内部精英怪系统。若启用 MythicMobs 集成，内部自然生成逻辑会跳过，由 MythicMobs 负责怪物内容。

| 精英怪 | 默认生成率 | 生命 | 攻击 | 主要特点 |
| --- | ---: | ---: | ---: | --- |
| 骷髅精英 | 12% | 30 | 5 | 使用生锈的铁剑，带中毒相关能力，并保留原版骷髅仇恨判定。 |
| 僵尸精英 | 12% | 35 | 5 | 使用亢奋石剑，提供近战压力。 |
| 精英蜘蛛 | 12% | 35 | 1.2 | 移速较高，带跳跃/位移相关能力参数。 |

精英怪数据和普通怪经验可通过 `content/mobs/*.yml` 调整。内置怪物 YAML 通过 `template:` 选择原版实体兜底，并用 `actions:` 组装动作块，例如 `target_close -> melee-burst -> retreat`。代码只实现 reusable action blocks，例如 `melee-burst`、`retreat`、`leap`、`shockwave`、`blink`、`blade-storm`，不再为具体怪物保留专名内容。

部分原版普通怪默认带有强化配置，例如尸壳、溺尸、掠夺者和僵尸猪灵会获得适度属性倍率和指定武器模板。服主可在 `content/mobs/<怪物 ID>.yml` 中用 `type: modifier` 调整或移除这些规则。

## 掉落内容

怪物随机武器掉落默认关闭。将 `gameplay.weapon-drop-multiplier` 从 `0.0` 调高后，怪物死亡时才会有低概率掉落内置武器模板。掉落优先判定高品质，命中某个品质后从该品质可掉落武器中随机选择一个。

| 品质 | 启用后基础概率 |
| --- | ---: |
| legendary | 0.2% |
| epic | 0.5% |
| rare | 1.0% |
| common | 2.0% |

`special` 品质默认不参与随机掉落。

## 展示与交互

### 玩家命令

| 命令 | 用途 |
| --- | --- |
| `/rl` | 查看玩家 Roguelike 状态。 |
| `/rl status` | 查看等级、经验、击杀、死亡等信息。 |

### 侧边栏

内置侧边栏默认开启，会显示：

- 玩家名。
- 当前等级。
- 当前等级内经验进度。
- 击杀数。
- 死亡数。
- 当前武器能力相关状态行。

如服务器使用 TAB 或其他展示插件，可以关闭内置侧边栏，并通过 PlaceholderAPI/TAB 做外部展示。

## 配置与扩展

首次启动后，插件会在 `plugins/Roguelike/` 下生成主要配置文件。

| 文件 | 作用 |
| --- | --- |
| `config.yml` | 存储方式、调试开关、外部插件集成、侧边栏设置。 |
| `weapons.yml` | 武器模板、基础属性、品质和默认效果。 |
| `items.yml` | 内置物品、防具套装物品定义。 |
| `mobs.yml` | 怪物经验、精英怪开关、精英怪属性。 |
| `forge-recipes.yml` | 铸造台配方。 |

修改配置后执行：

```text
/rw reload
```

可选集成包括 PlaceholderAPI、TAB、CommandAPI、MythicMobs 和 Nova。未安装这些插件时，Roguelike 会使用自身默认逻辑。

## 推荐游玩路线

1. 初期通过击杀普通怪、挖矿和吃东西升到 2 级，获得第一批开发券、强化券和移除券。
2. 使用开发券把常用武器或工具开发成特殊武器，进入 Roguelike 成长系统。
3. 继续刷怪和积累资源；若服务器显式开启插件掉落，可尝试获取内置武器掉落。
4. 根据武器方向选择保留词条，例如吸血续航、连锁群伤、暴击爆发或 Dash 机动。
5. 用强化券提升核心词条，用移除券清理不适合当前构筑的词条。
6. 建造铸造台，按服务器配方制作套装或特殊装备。
7. 中后期挑战更高经验怪物和精英怪，围绕套装、契约词条和高级武器形成构筑。
